Instruções Básicas

Para começar a usar o ElectronAPIManager, siga estas instruções básicas:

  1. Após iniciar a aplicação, você terá acesso a uma interface amigável para gerenciar dados por meio da API REST interna.
  2. Use as opções de menu ou botões na interface para visualizar, adicionar, editar ou excluir registros.
  3. Explore as funcionalidades de filtro e classificação para acessar os dados desejados com facilidade.
  4. Para compilar o projeto em um executável desktop, use o comando 'npm run make' no terminal.

Se você tiver dúvidas ou encontrar algum problema, entre em contato com o suporte.
